# 個人網站使用說明

這是一個簡單的個人網站，包含首頁、高中以前經歷、專案作品、活動參與和旅遊記錄等頁面。

## 檔案結構

```
hw1/
├── index.html          # 首頁
├── childhood.html      # 高中以前經歷
├── projects.html       # 專案作品
├── activities.html     # 活動參與
├── travel.html         # 旅遊記錄
├── styles.css          # 樣式檔案
└── README.md           # 說明文件
```

## 如何填入個人資訊

### 1. 首頁 (index.html)

在首頁中，你需要填入以下資訊：

- **個人照片**：在 `.photo-placeholder` 區域放入你的照片（建議尺寸：200x200px）
- **基本資料**：
  - 姓名
  - 年齡
  - 就讀學校
  - 科系
  - 聯絡方式（Email、電話、地址）
  - 興趣專長

### 2. 高中以前經歷 (childhood.html)

填入以下時期的資訊：

- **國小時期**：
  - 就讀學校
  - 重要回憶
  - 興趣發展

- **國中時期**：
  - 就讀學校
  - 學業表現
  - 課外活動
  - 重要經歷

- **高中時期**：
  - 就讀學校
  - 科系選擇
  - 學業成就
  - 社團參與
  - 重要活動
  - 未來規劃

### 3. 專案作品 (projects.html)

為每個專案填入：

- **專案基本資訊**：
  - 專案名稱
  - 專案簡介
  - 技術使用
  - 我的角色
  - 專案成果
  - 學習收穫

- **專案展示**：
  - 在 `.project-media` 區域放入專案截圖（建議尺寸：400x300px）

- **技能總覽**：
  - 程式語言
  - 網頁技術
  - 工具軟體
  - 其他技能

### 4. 活動參與 (activities.html)

填入以下類型的活動：

- **學術活動**：
  - 活動名稱、時間、地點
  - 主辦單位、我的角色
  - 活動內容、收穫與心得

- **社團活動**：
  - 社團名稱、參與時間
  - 擔任職務、主要活動
  - 重要經歷、學習成長

- **志工服務**：
  - 服務名稱、時間、地點
  - 服務內容、服務對象
  - 服務心得、影響與收穫

- **競賽活動**：
  - 競賽名稱、時間、地點
  - 競賽性質、參賽作品
  - 競賽結果、參賽心得

- **其他活動**：
  - 活動名稱、時間、性質
  - 參與方式、活動內容
  - 個人收穫

### 5. 旅遊記錄 (travel.html)

為每次旅遊填入：

- **基本資訊**：
  - 旅遊地點、時間、天數
  - 同行人員、旅遊方式

- **詳細內容**：
  - 行程安排（按天數）
  - 景點介紹
  - 美食體驗
  - 旅遊心得
  - 印象深刻的事

- **旅遊照片**：
  - 在 `.photo-placeholder` 區域放入照片（建議尺寸：300x200px）

- **旅遊統計**：
  - 旅遊次數、去過的城市
  - 最喜歡的地方、下次想去的地方

- **旅遊心得分享**：
  - 旅遊小貼士
  - 必備物品
  - 旅遊建議

## 如何修改網站

### 修改顏色主題

在 `styles.css` 檔案中，你可以修改以下顏色變數：

- 主要顏色：`#3498db`（藍色）
- 次要顏色：`#2c3e50`（深藍色）
- 強調顏色：`#e74c3c`（紅色）
- 成功顏色：`#27ae60`（綠色）
- 警告顏色：`#f39c12`（橙色）

### 添加新頁面

1. 建立新的 HTML 檔案
2. 在 `styles.css` 中複製現有的樣式
3. 在導航選單中添加新連結

### 修改版面配置

在 `styles.css` 中修改以下區域：

- `.profile-content`：個人資料區域的版面
- `.link-grid`：快速導覽的網格配置
- `.timeline`：時間軸的樣式
- `.project-content`：專案展示的版面

## 瀏覽網站

1. 用瀏覽器開啟 `index.html` 檔案
2. 點擊導航選單中的連結瀏覽不同頁面
3. 網站支援響應式設計，在手機和電腦上都能正常顯示

## 注意事項

- 所有需要填入的資訊都標示了 `TODO:` 註解
- 建議的圖片尺寸已標示在對應位置
- 可以根據實際情況增減專案、活動或旅遊的數量
- 記得定期備份你的個人資訊

## 技術支援

如果遇到問題，可以：

1. 檢查 HTML 語法是否正確
2. 確認 CSS 檔案路徑是否正確
3. 檢查圖片檔案是否存在
4. 使用瀏覽器的開發者工具查看錯誤訊息

祝你使用愉快！
