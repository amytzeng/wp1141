# 塔防遊戲

一個基於HTML5 Canvas的塔防遊戲，包含10個不同的關卡，每關都有獨特的地圖設計和敵人配置。遊戲具有完整的造型系統、商店系統和Boss戰鬥。

## 遊戲特色

- 🎮 **10個獨特關卡**：從基礎關卡到Boss關卡，每關都有獨特背景
- 🏗️ **3種塔防類型**：基礎塔(黃色)、快速塔(紫色)、重型塔(紅色)
- 🎨 **造型系統**：三角形(攻擊型)、正方形(射程型)、圓形(預設)
- 👾 **多種敵人**：基礎、快速、坦克、Boss敵人
- 🛒 **商店系統**：購買造型，永久擁有，提供屬性加成
- 🎵 **音效系統**：射擊、擊中、敵人死亡等音效
- 🎨 **視覺效果**：粒子效果、背景圖片、Boss動畫
- 📱 **響應式設計**：支援不同螢幕尺寸

## 遊戲玩法

### 基本規則
- 每關有5條生命值
- 怪物沿著道路移動到終點
- 每個到達終點的怪物會消耗1條生命
- 擊殺怪物獲得金幣，用金幣建造塔防
- 完成所有波次即可通關

### 塔防類型
1. **基礎塔** (20金幣) - 黃色
   - 傷害：20
   - 射程：80
   - 射速：1秒
   - 平衡型塔防，適合新手

2. **快速塔** (40金幣) - 紫色
   - 傷害：10
   - 射程：70
   - 射速：0.3秒
   - 高射速，適合清理小兵

3. **重型塔** (70金幣) - 紅色
   - 傷害：40
   - 射程：120
   - 射速：2秒
   - 高傷害，適合對付坦克敵人

### 造型系統
- **三角形造型**：大幅提升攻擊力，適合對付高血量敵人
- **正方形造型**：大幅提升射程，覆蓋更大範圍
- **預設造型**：圓形外觀，標準屬性，免費使用

### 敵人類型
1. **紅色敵人** (基礎) - 1金幣，10分
   - 生命值：70
   - 速度：中等
   - 最常見的敵人

2. **黃色敵人** (快速) - 3金幣，30分
   - 生命值：50
   - 速度：快
   - 移動速度快，血量較少

3. **藍色敵人** (坦克) - 5金幣，50分
   - 生命值：300
   - 速度：慢
   - 高血量，需要重火力

4. **Boss敵人** - 10-50金幣，100-500分
   - 生命值：1000-10000
   - 速度：極慢
   - 特殊外觀，極高血量

## 操作說明

### 滑鼠操作
- 點擊塔防選項選擇要建造的塔防類型
- 在草地上點擊放置塔防
- 無法在道路上建造塔防

### 鍵盤操作
- `空白鍵`：暫停/繼續遊戲
- `R鍵`：重新開始遊戲

### 按鈕操作
- **開始波次**：開始下一波敵人攻擊
- **暫停**：暫停/繼續遊戲
- **下一關**：完成關卡後進入下一關

## 關卡設計

### 前5關：基礎關卡
- **關卡1-5**：每關5波次，適合新手學習
- 獨特背景圖片，每關都有不同主題
- 逐步增加敵人難度和數量

### 後5關：進階關卡
- **關卡6-10**：每關5波次，包含Boss戰鬥
- 新增敵人類型：飛行敵人、裝甲敵人
- 新增塔防類型：冰霜塔、雷射塔
- 複雜的地圖設計，多條路徑

### Boss關卡
- **第10關**：最終Boss關卡
- 最後一波包含超強Boss敵人
- 特殊動畫效果和超高血量
- 完成後解鎖所有功能

## 商店系統

### 造型購買
- 使用商店金幣購買造型
- 商店金幣來源：關卡分數÷100
- 造型永久擁有，提供屬性加成

### 造型效果
- **三角形造型**：攻擊力+5到+15
- **正方形造型**：射程+12到+25
- **預設造型**：標準屬性，免費使用

## 技術特色

- **純前端實現**：無需後端服務器
- **HTML5 Canvas**：流暢的2D圖形渲染
- **Web Audio API**：動態音效生成
- **物件導向設計**：清晰的代碼結構
- **響應式布局**：適配不同設備

## 開始遊戲

### 方法1：直接開啟
1. 開啟 `index.html` 文件
2. 選擇關卡開始遊戲
3. 建造塔防防禦敵人
4. 完成所有關卡獲得勝利！

### 方法2：使用npm (推薦)
1. 安裝依賴：`npm install`
2. 啟動服務器：`npm start`
3. 在瀏覽器中開啟 `http://localhost:3000`
4. 享受更好的遊戲體驗！

## 開發說明

遊戲使用原生JavaScript開發，包含以下主要類別：
- `TowerDefenseGame`：主遊戲類
- `Tower`：塔防類
- `Enemy`：敵人類
- `Bullet`：子彈類
- `Particle`：粒子效果類
